# My homepage

Nothing to really put here. You can see it at [kmdw.dev](https://kmdw.dev).

The GitHub logo is from the
[GitHub brand toolkit](https://brand.github.com/foundations/logo) and I made
the cherry blossom tree animation myself.

## Development

The actual page is a Vite/Svelte app. Run the development server with 
`pnpm dev` and build it with `pnpm build`.

The Python part is basically there because before all the fancy GitHub stuff,
it used some backend stuff and it kinda just stayed there. It contributes
barely anything to the site itself. For deployment, it just serves file out of
`dist`.
